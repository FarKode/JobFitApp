## About JobFit

**JobFit** is an offline-first ATS Resume Score Checker & CV Builder for Android. Build professional resumes, check ATS compatibility scores, and export polished PDFs — all without an internet connection.

### Key Features

- 📝 Step-by-step resume builder with 7 sections
- 🎯 ATS keyword matching engine with 5,000+ industry keywords
- 📊 Real-time ATS compatibility score (0–100%)
- 🤖 AI-powered phrasing optimizer (offline)
- 💡 Smart skill-gap analyzer with role-based recommendations
- 📋 ATS format compliance checklist
- 🎨 5 professionally designed PDF templates
- 🌙 Dark mode support
- 🔒 100% offline — your data never leaves your device
